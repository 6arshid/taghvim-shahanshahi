# تقویم چندگانه

این مخزن شامل سورس‌کد یک **تقویم چندگانه وب‌اپلیکیشن** است که تاریخ‌ها را در قالب‌های متفاوت نمایش می‌دهد.

نسخه‌ی آنلاین این پروژه را می‌توانید از لینک زیر مشاهده کنید:

https://taghvim.netlify.app/

## درباره پروژه

وب‌اپلیکیشن «تقویم چندگانه» تاریخ روز جاری را در چند قالب مختلف نمایش می‌دهد:

- تاریخ **شاهنشاهی**
- تاریخ **هجری شمسی (Jalali)**
- تاریخ **هجری قمری (Hijri)**
- تاریخ **میلادی (Gregorian)**
- نمایش **روز هفته**

این ابزار می‌تواند برای نمایش سریع تاریخ‌ها در فرمت‌های مختلف مفید باشد و به‌عنوان یک نمونه پروژه‌ی ساده اما کاربردی در وب استفاده شود. :contentReference[oaicite:0]{index=0}

## ویژگی‌ها

- رابط کاربری ساده و قابل استفاده
- نمایش تاریخ برابر در چند تقویم مختلف
- مناسب برای یادگیری و توسعه با HTML/JavaScript/CSS
- قابل گسترش برای افزودن امکانات بیشتر
